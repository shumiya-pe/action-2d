# アセット一覧

このプロジェクトで使用しているアセットとそのライセンスの一覧。
**外部アセットを追加した場合は、必ずこのファイルに出典とライセンスを記載すること。**

## 現状

外部アセットは使用していない。すべて実行時にコードで生成したプレースホルダーである。

| アセット                                 | 生成方法                                                     | 備考                   |
| ---------------------------------------- | ------------------------------------------------------------ | ---------------------- |
| プレイヤー・敵・コインのテクスチャ       | Phaser の Graphics（`src/core/gfx/textures.ts`）で実行時生成 | 矩形・円のみ           |
| 足場・ゴール旗                           | `Phaser.GameObjects.Rectangle` / `Triangle`                  | テクスチャ不使用       |
| PWA アイコン（`public/icons/`）・favicon | `scripts/generate-icons.mjs` で生成（`npm run icons`）       | 外部依存なしの自作 PNG |
